<strong>Spring boot crud application </strong> </br>
API endpoint
</br>
GET /courses
</br>
Get all courses
</br>
POST /courses/add
</br>
add one course
</br>
GET /courses?key=option&value=yourvalue
</br>
Get course by option.</br>
Option available:
<ul>
<li>id</li>
<li>title</li>
<li>description</li>
</ul>
